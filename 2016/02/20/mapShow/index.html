简介 介绍在百度地图上利用 label 实现圆形标识，添加轨迹和 marker 移动效果的实现方法。效果展示点这里。其中涉及百度地图API的相关内容以及js中 setTimeout() 和 setInterval() 函数的使用。

创建并初始化地图

div+css部分

首先，加入一个id 为”map”的 div。并设置它的css样式，这里为了简便，就直接用”style”设置了。

<div id="map" style="height:600px;width:100%;"></div>

js部分

初始化地图主要是以下三句：

var map = new BMap.Map("map",{minZoom:18,maxZoom:19});   // 创建地图实例
var pt_center = new BMap.Point(116.364506, 39.969775);//创建一个点
map.centerAndZoom(pt_center,19 );         // 初始化地图，设置中心点坐标和地图级别

百度地图相关标识如何实现

利用label实现圆形标识

这里，圆形标识主要用来表示一个地点，标识中有数字来表示这个位置的人数。百度地图自定义 marker 比较复杂也不灵活，所以这里我们用 label 来实现。

var myLabel = new BMap.Label("<p>"+data.N[j][i]+"</p>",     //为lable填写内容
{
    position: new BMap.Point(lonlat[i].Longitude,lonlat[i].Latitude)});  //label的位置
myLabel.setStyle({       //给label设置样式，任意的CSS都可以
      height:"24px" ,         //高度
    width:"24px",           //宽
    borderRadius:"12px",    //边框圆角半径，设置为边长的一半就变成圆了
    opacity:"0.8",            //透明度
    border:"0",             //边框设置成无
    color:”black",            //文字颜色
    fontSize:"22px",        //字号
    textAlign:"center",     //文字水平居中显示
    lineHeight:"20px",      //行高，设置文字位置
});

lonlat[i]存放了表示从第i个地点的经纬度信息。data.N[j][i]表示第j次人数变化后第i点的人数。

这里需要一个效果，地点人数不同的时候，这个圆形标识的颜色不同，所以我们需要使用if语句来判断人数的范围：

if(data.N[j][i]<a0){ 
    myLabel.setStyle({background:"#ffcc00", }); }

这样我们就可以得到一个下图这样带有数字的圆形标识啦~

关于label你想知道更详细的内容，可以参考这篇博文：【百度地图API】——如何用label制作简易的房产标签

添加轨迹和自定义图标marker

添加轨迹利用百度地图的 BMap.Polyline(point0，point1) 即可。代码如下：

var polyline = new BMap.Polyline(new BMap.Point(lonlat[i].Longitude+lon, lonlat[i].Latitude-lat),new BMap.Point(lonlat[k].Longitude+lon, lonlat[k].Latitude-lat)], {    
    strokeColor:"#FFCC00",         //轨迹线的颜色
     strokeWeight:3,             //轨迹线的粗细
    strokeOpacity:0.8});        //轨迹线的透明度
map.addOverlay(polyline);

lon 和 lat 都是为了让轨迹从圆形标识的中间出来而设置的偏移量。

为了在轨迹上用 marker 的移动来表示人员移动方向，我们需要添加自定义图标。因为我们需要在图标上用数字表示移动人员数，所以这里也选择用 label 来实现。代码如下：

var marker= new BMap.Label("<p>"+data[i][k]+"</p>",     //为lable填写内容
{    offset:new BMap.Size(-16,-16),         //label的偏移量
    position:new BMap.Point(lonlat[i].Longitude+lon+d_lon,lonlat[i].Latitude-lat+d_lat)    //label的位置
});
marker.setStyle({                                  
    fontSize:"14px",               //字号
    border:"0",                    //边
    height:"25px",                //高度
    width:"25px",                 //宽
    textAlign:"center",            //文字水平居中显示
    background:"url(img/marker.png)",    //自定义图标的图片
    color:"blue",        //文字颜色
    fontWeight:"bold",        //文字加粗
});
map.addOverlay(marker);

让你的标识动起来

setTimeout()和setInterval()

setTimeout() 用于在指定的毫秒数后调用函数或计算表达式（只运行一次）。而 setInterval() 可按照指定的毫秒数周期性地调用函数或计算表达式。但是 setTimeout() 方法也可以用于循环执行的——当函数自身再次调用 setTimeout() 的时候。

关于这两个函数的作用方法，我们知道，JavaScript是以单线程方式运行于浏览器的JavaScript引擎中的，setTimeout() 和 setInterval() 会把要执行的代码在设定的时间点插入js引擎维护的一个代码队列中，所以时间到了代码并不一定就会执行。

我们这里利用这两个函数来实现间隔一定时间动态展示的功能。篇幅限制，不展开介绍了~关于它们的定义、使用例子可以参考javascript的setTimeout()用法总结，js的setTimeout()方法，关于它们的作用方式可以参考setTimeout和setInterval的区别你真的了解吗?。百度or google 一下可以找到很多资料哦~

让你的自定义图标动起来

自定义图标的移动方向表示了人员的移动方向。这里我们使用 setInterval() 函数来实现。

首先，我们在前面的代码中要进行初始化设置，和传递参数的工作。

var movei=0;        //图标移动次数
var moveTime= window.setInterval(function(){delayMove(data.M[j-1]);}, 500);

接下来，就是函数里的一些设置了。data[i][k] 中存放了从地点i转移到地点k的人数。 d_lon 和 d_lat 表示图标向着终点偏离起始点的经纬度，为了图标每次显示位置变化而设置。使用全局变量数组 moveMarker 存放此次的 marker，到下次 marker 位置变化的时候就清除掉。

var moveMarker=new Array();        //为了下次显示的时候清除标识而设置
function delayMove(data){
    var movei_max=8;    //移动的次数
    lon=0.00015/32*12;    //偏移量
    lat=0.00011/32*12;    
    if(movei<movei_max){
        var markerNum=0;
        for(var i=0;i<lonlat.length;i++){
            for(var k=0;k<lonlat.length;k++){ 
                if(data[i][k]!=0){                                  
                    map.removeOverlay(moveMarker[markerNum]);    //清除前次的图标    
                    var d_lon=(lonlat[k].Longitude-lonlat[i].Longitude)/(movei_max)*(movei+1);
                    var d_lat=(lonlat[k].Latitude-lonlat[i].Latitude)/(movei_max)*(movei+1);
                    var marker= new BMap.Label("<p>"+data[i][k]+"</p>",     //为lable填写内容
                    {        
                        offset:new BMap.Size(-16,-16),                  //label的偏移量，为了让label的中心显示在点上
                        position:new BMap.Point(lonlat[i].Longitude+lon+d_lon,lonlat[i].Latitude-lat+d_lat)});                                //label的位置
                    marker.setStyle({                                   //给label设置样式，任意的CSS都是可以的
                        fontSize:"14px",               //字号
                        border:"0",                    //边
                        height:"25px",                //高度
                        width:"25px",                 //宽
                        textAlign:"center",            //文字水平居中显示
                        //lineHeight:"20px",            //行高，文字垂直居中显示
                        background:"url(img/marker.png)",    //背景图片
                        color:"blue",
                        fontWeight:"bold",
                    });
                    moveMarker[markerNum]=marker;
                    markerNum++;
                    map.addOverlay(marker);    
                }
            }
        }
        movei++;
    }else{
        var markerNum=0;
        for(var i=0;i<lonlat.length;i++){
            for(var k=0;k<lonlat.length;k++){ 
                if(data[i][k]!=0){                        
                    map.removeOverlay(moveMarker[markerNum]);
                    markerNum++;
                }

            }
        }
        movei=0;
        window.clearInterval(moveTime);
    }
}

变换你的图

当人员从一个地点移动到另一个地点，相应地，圆形标识中的数字也变化了，而且要准备下次人员移动的轨迹和动态方向的显示，这里为了不让用户等待一段时间才出现标识，我们不再使用 setInterval() 函数，而使用setTimeout()函数来实现。在显示自定义标识的 delay 函数中每一次自定义标识动态显示完毕后，加一句

TimeID = window.setTimeout(function(){delay(data);}, 5000);

这样就可以间隔一定时间显示一次delay函数，也就是再来一次显示圆形标识和相应人数，显示人员移动状况啦~

总结
