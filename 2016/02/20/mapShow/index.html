简介 介绍在百度地图上利用 label 实现圆形标识，添加轨迹和 marker 移动效果的实现方法。效果展示点这里。文章主要分为两部分，一部分介绍用到的百度地图相关功能，一部分讲解动态变化的实现以及暂停等功能的实现，主要涉及 js 中 setTimeout() 函数的使用 。

初始化工作

要用到的数据

为了方便理解程序，首先介绍要用到的数据。这里，我们使用lonlat数组来存储地点的经纬度信息。data.N来存储地点的人数信息，比如data.N[j][i]就表示第j次变化时第i个地点的人数。data.M来存储移动的人数，比如data.M[j][i][k]表示第j次变化时第i个地点向第k个地点转移的人数。

创建并初始化百度地图

首先，加入一个id 为”map”的 div。并设置它的css样式，这里为了简便，就直接用”style”设置了。

<div id="map" style="height:600px;width:100%;"></div>

js中初始化地图主要是以下三句：

var map = new BMap.Map("map",{minZoom:18,maxZoom:19});   // 创建地图实例
var pt_center = new BMap.Point(116.364506, 39.969775);//创建一个点
map.centerAndZoom(pt_center,19 );         // 初始化地图，设置中心点坐标和地图级别

在百度地图上添加地点标识和移动符号标识

利用label实现圆形标识

这里，圆形标识主要用来表示一个地点，标识中有数字来表示这个位置的人数。百度地图自定义 marker 比较复杂也不够灵活，所以这里我们用 label 来实现。

var myLabel = new BMap.Label("<p>"+data.N[j][i]+"</p>",     //为lable填写内容
{
    position: new BMap.Point(lonlat[i].Longitude,lonlat[i].Latitude)});  //label的位置
myLabel.setStyle({       //给label设置样式，任意的CSS都可以
      height:"24px" ,         //高度
    width:"24px",           //宽
    borderRadius:"12px",    //边框圆角半径，设置为边长的一半就变成圆了
    opacity:"0.8",            //透明度
    border:"0",             //边框设置成无
    color:”black",            //文字颜色
    fontSize:"22px",        //字号
    textAlign:"center",     //文字水平居中显示
    lineHeight:"20px",      //行高，设置文字位置
});

这里用到了每个地点的经纬度信息和人数信息。

我们需要一个效果——地点人数不同的时候，圆形 label 的颜色不同，所以我们需要使用if语句来判断人数的范围：

if(data.N[j][i]<a0){ 
    myLabel.setStyle({background:"#ffcc00", }); }

这样我们就可以得到一个下图这样带有数字的圆形标识啦~

关于label你想知道更详细的内容，可以参考这篇博文：【百度地图API】——如何用label制作简易的房产标签.

添加轨迹和自定义图标marker

添加轨迹利用百度地图的 BMap.Polyline(point0，point1) 即可。代码如下：

var polyline = new BMap.Polyline(new BMap.Point(lonlat[i].Longitude+lon, lonlat[i].Latitude-lat),new BMap.Point(lonlat[k].Longitude+lon, lonlat[k].Latitude-lat)], {    
    strokeColor:"#FFCC00",         //轨迹线的颜色
     strokeWeight:3,             //轨迹线的粗细
    strokeOpacity:0.8});        //轨迹线的透明度
map.addOverlay(polyline);        //在地图中加入轨迹

lon 和 lat 都是为了让轨迹从圆形 label 的中间出来而设置的偏移量。

为了在轨迹上用 marker 的移动来表示人员移动方向，我们需要添加自定义图标。因为我们需要在图标上用数字表示移动人员数，所以这里也选择用 label 来实现。代码如下：

var marker= new BMap.Label("<p>"+data.M[j][i][k]+"</p>",     //为lable填写内容
{    offset:new BMap.Size(-16,-16),         //label的偏移量
    position:new BMap.Point(lonlat[i].Longitude+lon+d_lon,lonlat[i].Latitude-lat+d_lat)    //label的位置

marker.setStyle({

让你的图动起来

setTimeout()和setInterval()

setTimeout() 用于在指定的毫秒数后调用函数或计算表达式（只运行一次）。而 setInterval() 可按照指定的毫秒数周期性地调用函数或计算表达式（第一次调用也是指定的毫秒数后）。但是 setTimeout() 函数也可以用于循环执行的——当函数自身再次调用 setTimeout() 的时候。

关于这两个函数的作用方法，我们知道，JavaScript是以单线程方式运行于浏览器的JS引擎中的，所以要以单线程的方式理解它，setTimeout() 和 setInterval() 会把要执行的代码在设定的时间点插入js引擎维护的一个代码队列中，所以时间到了如果前面的代码没有执行完它并不一定就会执行，而假如setTimeout()引入循环的话，这句后面的语句会执行之后再过一段时间来执行这个循环。比如：

var j=0;
function a(){
    var time=new Date().getMinutes()+":"+new Date().getSeconds();
    console.log("before time:",time,"j:",j);
    if(j<1){
        var timeid=setTimeout(function(){a()},2000);
        j++;console.log("after:time:",time,"j:",j);}
    else{clearTimeout(timeid);}
}
a();

这段程序console中呈现的结果是：

before time: 33:39 j: 0
after time: 33:39 j: 1
before time: 33:41 j: 1

也就是说j++;这句是直接跟 setTimeout() 后面马上执行的。相信这样你能更了解到 setTimeout() 函数如何作用的了吧~

我们这里利用setTimeout()函数来实现动态展示的功能。篇幅限制，关于它们的定义、使用例子可以参考javascript的setTimeout()用法总结，js的setTimeout()方法，关于它们的作用方式可以参考setTimeout和setInterval的区别你真的了解吗?。百度or google 一下可以找到很多资料哦~

动起来!

可以看到，这里我们使用了两次 setTimeout() 函数来实现动态变化。外循环是各个地点人数和地点间轨迹间隔一段时候后刷新，内循环是轨迹上的 marker 一次次变换位置。为了逻辑更清晰，下面用伪代码形式展现：

首先，我们要进行初始化设置：

j = 0;//第j次刷新图
movei=0;//marker移动次数
var TimeID=null;//外层setTimeout函数ID
var moveTime=null;//内层setTimeout函数ID

实现外循环的函数叫delay(),实现内循环的函数叫delayMove()。首先进入delay()
函数，为了清晰，下面就用伪代码表示结构啦。

function delay(){
    if(j<data.N.length)//总转移次数内
    {     在地图上显示每个地点的圆形label；
        判断地点之间是否有转移，有则画轨迹；
        window.clearTimeout(moveTime);
        if(j<data.N.length-1){
            moveTime= window.setTimeout(function(){delayMove();}, 500)
        }    //在最后一次显示之前都有轨迹要显示marker移动效果            
        TimeID = window.setTimeout(function(){delay();}, 5000);
            //注意这两个函数设置的时间，第二个应大于第一个乘以marker移动次数
    }
    else{ //显示完毕后，清除定时器
        window.clearTimeout(moveTime);
       j = 0;
       window.clearTimeout(TimeID);
     }    
}

接下来，deayMove函数结构如下：

var moveMarker=new Array();//为了清除上次地图上的marker而设置
function delayMove(){
    var movei_max=8;//最大的移动次数
    if(movei<movei_max){
        清除moveMarker所存的marker；
        给每条轨迹加上marker；
        moveMarker用这次的各个marker覆盖上次的；
        }
        movei++;
        moveTime= window.setTimeout(function(){delayMove();}, 500);    
    }else{//大于最大移动次数后
        在地图上清除moveMarker；
        j++;
        movei=0;
        window.clearTimeout(moveTime);
    }
}

我们注意到，上面j++;这句写在了每次marker移动完毕内循环结束后，这是因为上面我们讲到的在setTimeout函数后面执行的语句会马上执行，所以把它放到这里，避免混乱~

开始、暂停和重新开始功能

我们使用两个按键，一个id为”stop”的按键表示暂停，按下后变成开始。一个id为”start”的按键表示重新开始。首先，介绍”stop”的功能：

document.getElementById("stop").addEventListener("click",function(){
    var btn=document.getElementById("stop");
    if(btn.value=="暂停"){            
        btn.value="开始";
        //暂停清除定时器，但是全局变量j和movei的值不要清除，记录暂停位置
          window.clearTimeout(TimeID);
        window.clearTimeout(moveTime);
    }else{
        btn.value="暂停";
        var delayTime=500;
        //movei=0进入要先进入下次delay()，delay()中再调用delayMove()
        if(movei!=0){
            moveTime= window.setTimeout(function(){delayMove();}, 500);
            delayTime=500*(10-movei);        
        }
        //如果是演示结束后按"开始"就重新开始
        if(j==(data.N.length-1)&&movei==0)    {getMove()}
        else{//否则进入delay()
        TimeID = window.setTimeout(function(){delay();},delayTime);    
    }
    }
});

重新开始功能就比较简单了：

document.getElementById("start").addEventListener("click",function(){    
    document.getElementById("stop").value="暂停";
  getMove();
});
